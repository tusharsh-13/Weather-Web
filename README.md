# Weather Web

A modern, responsive weather web app built with HTML, CSS, and JavaScript. It shows real-time weather data, a 5-day forecast, sunrise/sunset times, and includes a smart city search — all wrapped in a glassmorphism UI.

This project was created as Task 3 for an internship.

---

## Features

- Real-time current weather: temperature, feels-like, humidity, wind speed, condition (with icon) and accurate local time.
- "Updated X minutes ago" indicator for freshness of data.
- 5-day forecast with min/max temperatures and daily icons (horizontal scroll UI).
- Sunrise and sunset times for the selected city.
- Unit toggle: switch between °C and °F instantly.
- Smart city search with common-alias normalization (e.g. "Delhi" → "Delhi, India").
- Quick-select buttons for popular cities.
- Clean, modern glassmorphism UI with responsive layout.

## Tech Stack

- HTML5
- CSS3
- JavaScript 
- WeatherAPI (https://www.weatherapi.com)
